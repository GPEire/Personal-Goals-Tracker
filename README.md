# Personal Goals Tracker

Iteration 1 implementation of the Personal Resolution Tracker (2026) MVP.

## Included in Iteration 1
- Frontend scaffold as installable PWA (React + Vite) with 3 tabs: Today, Goals, History.
- Backend scaffold (FastAPI) with auth and goal CRUD endpoints.
- Postgres schema and Alembic migration for users, goals, logs, and reminder jobs.
- Basic goal management flow (create/list/delete) in the Goals tab.

## Repo structure
- `frontend/` – PWA client.
- `backend/` – FastAPI API + migrations.
